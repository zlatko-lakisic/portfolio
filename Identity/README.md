# Identity & Access Architecture

[← Back to Main Portfolio](../index.md)

Identity and access work spans **enterprise federation** in client delivery and **zero-trust segmentation** in hands-on infrastructure — the same trust-boundary thinking applied at OAuth scale and at the VLAN level.

## Specialization

- **Enterprise federation** — SAML/OAuth SSO, identity-aware integration patterns for ECM and line-of-business platforms
- **Credential governance** — environment-scoped service catalogs for AI backends and MCP tool servers
- **Segmented trust zones** — explicit network boundaries between perimeter compute, IoT/OT, and management planes
- **Secure-by-default access** — least-privilege tool exposure and deliberate ingress rather than flat networks
